## Find transactions by tag

You may retrieve transactions via tag name/value pairs.

### Single Tag

To look up transactions by a single tag, you may structure your query as follows:

:::tip
Omitting the square brackets is permissible if you specify only one value per tag pair
:::

```graphql
query {
    transactions(tags: { name: "App-Name", values: ["test"] }) {
        edges {
        node {
            id
        }
      }
    }
}
```

### Multiple Tags

Alternatively, look up transactions using multiple tags like so:

```graphql
query {
    transactions(
        tags: [
            { name: "App-Name", values: ["Test"] }
            { name: "Content-Type", values: ["text/html"] }
        ]
      ) {
          edges {
              node {
                  id
              }
          }
    }
}

```
